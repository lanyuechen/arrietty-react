## Flex布局组件

- 布局通过flex实现，支持持现代浏览器
- 支持水平多栏布局和垂直多栏布局
- 多个栏目间可以通过size指定大小，不指定的自动填充剩余区域
- 复杂布局通过嵌套实现

| 属性 | 类型 | 描述 | 默认值 | 是否必须 |
|----|----|----|----|----|
| vertical | bool | 子元素布局方式，true：子元素垂直布局，false：子元素水平布局 | false | 否 |
| size | number\|string | 子元素尺寸，与vertical相关，vertical为真，则size表示高度 | '100%' | 否 |
| 其它 |
